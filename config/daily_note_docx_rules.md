# Daily note DOCX update rules

- 월요일부터 금요일까지 같은 주의 DOCX 파일 1개를 갱신한다.
- 기존 Word 템플릿의 안내문, 표, 머리글, 바닥글, 스타일은 유지한다.
- 각 날짜별 Markdown 연구노트가 있으면 해당 내용을 반영한다.
- 아직 생성되지 않은 날짜의 연구노트는 미생성 상태로 표시한다.
- 산출 파일명은 `{yyyymmdd~yyyymmdd}_daily_note.docx` 형식을 사용한다.

## 추가 업데이트 규칙

- 템플릿을 유지하며 날짜별 페이지를 생성한다.
- 날짜별 페이지는 항상 1페이지를 넘기지 않는 내용으로 요약 정리한다.
- 날짜별 페이지가 생성되면 Contents of Study에 쪽 번호와 연구 목적을 추가한다.
